# Groupomania social network

## Initialization
◼️ Install all dependencies
```
npm install
```
◼️ Install database   
Open a terminal, and run your mysql database
```
mysql -u root -p
```
Run the databse.sql source file
```
source [the absolute path of back/database.sql]
```

## Run servers 
◼️ To start front  
```
npm run front
```
◼️ To start back   
```
npm run back
```

Enjoy.
